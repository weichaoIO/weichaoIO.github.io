SketchUp Pro 2022 根据户型图建模

本文最后更新于：4 天前

导入户型图

文件 -> 导入 -> 选择图片

放大到正确的尺寸

卷尺工具测量两个点的尺寸 -> 输入长度为户型图中的尺寸

安装 Dibac 插件

扩展程序 -> 扩展程序管理器 -> 安装扩展程序 -> 选择本地下载好的 dibac.rbz

设置 Dibac 为中文

扩展程序 -> Dibac -> Settings -> Select language -> 选择cn -> 重启 SketchUp

Dibac

图标名称功能tips
墙体绘制墙壁点对点连续绘制时切换连接位置：tab
并联墙绘制墙平行的边缘
延伸墙体延长/缩短墙壁
门
窗放置在墙壁的窗口
柜体放置在墙壁柜
Choose joinery有一些门洞、窗洞等组件
楼梯从点对点或通过路径的绘制楼梯宽度=楼梯宽度
窗楣=踏步高度
起跑方向=踏步宽度
convert to 2D/3D二维/三维转换

开始绘制

绘制墙体

使用墙体、并联墙、延伸墙体。

绘制墙体时，把窗户、门、垭口都先用墙体绘制。

确定户型图中的墙厚

使用卷尺工具测量，一般是 120mm、240mm。

绘制门

使用门。

不确定尺寸时，可以先使用移动把模型整体移动。

绘制垭口

Choose joinery -> dibac_door_opening.skp

绘制窗洞、飘窗

Choose joinery -> dibac_window_opening.skp

绘制完成后创建群组

二维/三维转换

使用 convert to 2D/3D。

三维默认高度是 3000mm。

修改整体高度

使用推/拉。

完成
